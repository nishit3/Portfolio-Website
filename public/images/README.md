## Guidelines

### Project Images
- **ProStaff**: Add 3+ images showing HR dashboard, employee management, salary payment UI, architecture diagram
- **EarthAlly**: Add 3+ images showing IoT sensors, AWS dashboard, mobile app
- **Bus Transport System**: Add 3+ images of facial recognition, tracking interface, bus system
- **Kisan-Sevak**: Add 3+ images of farming monitoring, IoT devices, mobile interface
- **Neuronify**: Add 3+ images of neural network training UI, no-code interface

### Achievement Images
- **SAP-GTU**: Add 3+ images of hackathon, team photo, presentation, award ceremony
- **NASSCOM**: Add 3+ images of competition, team, award, project demo
- **Hackout 2023**: Add 3+ images of hackathon, team photo, award ceremony, project showcase
- **IEEE ICCE**: Add 3+ images of conference, presentation, paper screenshots
- **IEEE ICCCNT**: Add 3+ images of conference, presentation, paper screenshots

## Image Specifications
- Format: JPG, PNG, or WebP
- Recommended size: 1200x800px (3:2 aspect ratio)
- Max file size: 500KB per image (optimize for web)
- Quality: 80-85% for JPG

## Tips
- Use high-quality, professional photos
- Ensure images are well-lit and clear
- Include screenshots of working projects
- Add team photos from hackathons/conferences
- Compress images before adding to reduce load time

## Placeholder Images
Until you add your own images, the portfolio will use default placeholder images.
